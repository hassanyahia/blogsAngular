About Blog:

Blog is a free platform for blogging.

Features-

BLOGS

  • Write a blog with a rich text editor
  • Autosave blog while writing
  • Continue where you left even if you close the browser.
  • Update blog anytime after publishing.
  • Improve the reachability of every blog using categories and tags.

USERS

  • Create and update public profile.
  • View all blogs written by you in your dashboard.

BROWSING

  • View all latest blogs on the home page.
  • Filter by category from the home page.
  • Tap on any tag in a blog to view blogs having that tag.
  • Related blogs for each blog in the single blog page.
  • Commenting system by Disqus.
  • Search blogs with keywords.
  • Visit the profile page of any user to view all the blogs written by them.
  • Share blogs and user public profile directly.

THE WEB APP

  • Blogaroo is a Progressive Web App (PWA).
  • Can be installed as a standalone app.
  • Improved caching for faster rendering.
